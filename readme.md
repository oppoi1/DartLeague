# DartLeague

A website to start a Dart League.

Made with ExpressJS, GraphQL and VueJS.

## Project setup in both folder
```
npm install
```

## server

## Start server
```
npm start
```

## client

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
